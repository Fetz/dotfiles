# Karabiner custom keys

## symbols
- ⌘ → command / cmd
- ⌥ → alt / option
- ⇧ → shift
- ⎈ → control / ctrl / ctl
- ⇪ → caps lock

## apple internal keyboard

### In all applications
- `right ⌘` →  `left ⌥ + left ⇧ + left ⎈`
- `⌥ + right ⌘` →  `left ⌥ + left ⇧ + left ⎈ + left ⌘ `

### in terminal:
- `⇪` → `⎈ + a`
- `left ⇧ + ⇪` → `⇪`
